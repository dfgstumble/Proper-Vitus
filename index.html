What is a virus
A virus is a piece of program is a programme or a part of a programme developed by a corrupt programmer or virus author, who made that program to test it's payload (a destructive event a virus is thought to deliever). Types of Virus
File Infector: This type of virus add virus code to the regular programming code. So when this file is run, this infects other files
Boot Sector Virus: This type of virus infects the boot sector of the disk, which is read every time the computer is started up. This contains the file-allocation tables and start-up instructions. This virus is also dangerous
Executable Virus: This virus infects executable files (files ending with .exe)
Macro Virus: This type of virus infects programmes made by applications which use macros. These macros help in automating the operations, so the user doesn't have to do the operations one by one
